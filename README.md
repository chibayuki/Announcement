# 指南

## 开始之前
* Com是一个支持重要基础功能的DLL，你很可能需要它。
* 运行或编译我们的程序时，你可能会看到缺少"Com"程序集的提示。
* 在获取Com程序集之前，你需要知悉所需的正确版本，请参阅[Com兼容性报告](https://github.com/chibayuki/ComCmptReport)。
* 最后，请获取[Com](https://github.com/chibayuki/Com)。
